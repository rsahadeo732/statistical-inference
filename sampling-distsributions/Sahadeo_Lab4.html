10. Use data from Q3 as the initial sample, use bootstrapping method to resample once with 200 movies. Are there any duplicated movies in your bootstrap sample? Is this expected or something is wrong?

bootstrap_sample <- sample_200 %>%
  rep_sample_n(size = 200, replace = TRUE)

duplicate_movies <- sum(duplicated(bootstrap_sample))

duplicate_movies
## [1] 77

Yes there are 77 duplicated movies in my bootstrap sample. This is expected since bootstrapping is used to approximate sampling variability. Each re sampled data set is drawn from the original sample with replacement, meaning some movies appear multiple times while others may not appear at all. If we didnt sample with replacement we would just reshuffle the original sample.
